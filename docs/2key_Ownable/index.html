Ownable

contract Ownable

The Ownable contract has an owner address, and provides basic authorization control functions, this simplifies the implementation of "user permissions".

Source: 2key/Ownable.sol

    The Ownable constructor sets the original `owner` of the contract to the sender account.

  • renounceOwnership

    function renounceOwnership() public

    Allows the current owner to relinquish control of the contract., Renouncing to ownership will leave the contract without an owner. It will not be possible to call the functions with the `onlyOwner` modifier anymore.

    Modifiers:
    onlyOwner
