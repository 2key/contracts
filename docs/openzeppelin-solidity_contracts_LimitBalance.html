LimitBalance

contract LimitBalance

Simple contract to limit the balance of child contract. Note this doesn't prevent other contracts to send funds by using selfdestruct(address); See: https://github.com/ConsenSys/smart-contract-best-practices#remember-that-ether-can-be-forcibly-sent-to-an-account.

Source: openzeppelin-solidity/contracts/LimitBalance.sol
