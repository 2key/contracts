  • addContract

    function addContract(address _contractAddress) external

    Only admin can call this or an authorized person, function where admin or any maintainer can add more contracts to allow them call methods.

    Modifiers:
    onlyMaintainerOrTwoKeyAdmin
    Parameters:
    _contractAddress - is actually the address of contract we'd like to allow

    function addMaintainers(address[] _maintainers) external

    Only twoKeyAdmin contract is eligible to mutate state of maintainers, Function which can add new maintainers, in general it's array because this supports adding multiple addresses in 1 trnx.

    Modifiers:
    onlyTwoKeyAdmin
    Parameters:
    _maintainers - is the array of maintainer addresses
