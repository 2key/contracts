  • verifyProof

    function verifyProof(bytes32[] _proof, bytes32 _root, bytes32 _leaf) internal pure returns (bool)

    Verifies a Merkle proof proving the existence of a leaf in a Merkle tree. Assumes that each pair of leaves and each pair of pre-images are sorted.

    Parameters:
    _proof - Merkle proof containing sibling hashes on the branch from the leaf to the root of the Merkle tree
    _root - Merkle root
    _leaf - Leaf of Merkle tree
    Returns:
    bool
