MaintainingPattern

contract MaintainingPattern

This is maintaining pattern supporting maintainers and twoKeyAdmin as ``central authority`` which is only eligible to edit maintainers list.

Source: contracts/2key/MaintainingPattern.sol
Author: Nikola Madjarevic
